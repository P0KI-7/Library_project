Система управления библиотекой на C++

## 1. Описание проекта
Консольное приложение для управления личной библиотекой с использованием объектно-ориентированного программирования. Приложение демонстрирует практическое применение основных концепций ООП: классы, объекты, инкапсуляция, наследование и полиморфизм.

## 2. Требования
C++17 или новее
CMake 3.10 или новее
Компилятор с поддержкой C++17 (GCC, Clang, MSVC)
Система сборки (Make, Ninja и т.д.)


## Инструкции по сборке
1. Клонирование репозитория
git clone <rhttps://github.com/P0KI-7/Library_project.git> (без <>)
cd library_project

2. Сборка с помощью CMake
mkdir build
cd build
cmake ..
make

3. Запуск приложения
./library
